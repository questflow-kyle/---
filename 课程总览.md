# 赤耘计算机通识课

## 课程定位

面向会基本电脑操作的学员，统一培养目标：**每个人都能用AI进行开发**。通识内容（搜索、办公、硬件、安全）保留但压缩，为AI开发能力让出课时。

## 基本信息

| 项目 | 内容 |
|:---|:---|
| 归属 | 培训部 · 通识教育组 |
| 讲师团队 | 梅杭俊、沈林伟、尹伟 |
| 授课形式 | 线下面授，隔天一节 |
| 课时标准 | 每节课 45 分钟 |
| 学员管理 | 5-10 人小班 |
| 学员起点 | 会基本电脑操作（开机、浏览器、打字） |
| AI工具策略 | 基础阶段用网页版AI → 实践阶段引入Claude Code |

## 课程结构

| 阶段 | 节数 | 内容 |
|:---:|:---:|:---|
| 基础课 | 第1-8节 | 通识压缩 + 编程入门 + AI协作开发基础 |
| 实践课 | 第9-13节 | 5个递进式开发项目，同任务不同深度 |
| 考核 | 第14节 | 作品展示 + 答辩 |
| **合计** | **14节** | |

## 讲师分工

| 讲师 | 专业背景 | 主讲内容 |
|:---:|:---|:---|
| 梅杭俊 | 非技术背景，丰富AI实战经验 | 搜索、AI对话、办公（第1-3节） |
| 沈林伟 | 专业AI agent软件工程师 | 编程、Claude Code、开发实战（第5-13节） |
| 尹伟 | 电源行业软件工程师，偏嵌入式 | 硬件认知、环境搭建、部署（第4节，实践课辅助） |

## 考核等级

| 等级 | 关键词 | 标准 |
|:---:|:---:|:---|
| L1 | **能用** | 能用AI进行开发工作；能通过搜索、提问解决大部分问题；了解计算机开发基础 |
| L2 | **能调** | 能理解AI生成的代码；能有方向地调整代码逻辑和功能 |
| L3 | **能交付** | 能完整走通开发→调试→部署全流程；能独立交付可运行的成品 |

- **L2 是期望线**：所有学员应达到
- L1 是保底，L3 是优秀
- L3"部署"定义视项目类型而定：脚本类本地运行即可，应用类需上线可访问
