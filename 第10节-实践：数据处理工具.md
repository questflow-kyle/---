# 第10节：实践——数据处理工具

## 概要

> 在第7节基础上升级：从"处理一个CSV"到"做一个完整的数据处理工具"。任务场景贴近真实工作——处理考勤数据、生成工资单。比上一节复杂度提升：需要多步骤处理、多文件输入输出。
>
> **待补充**：
> - [ ] 准备模拟考勤数据CSV（20-30条记录，含正常/迟到/缺勤/加班）
> - [ ] 准备工资参数表（基本工资、加班费率、迟到扣款等）
> - [ ] 确认openpyxl是否需要预装（如涉及Excel输出）

## 教案

### 课时信息

- 所属阶段：实践课（第2节/共5节）
- 主讲：沈林伟
- 辅助：梅杭俊
- 时长：45 分钟

### 教学目标

学完本节课，学员能够：
1. 用Claude Code开发一个多步骤的数据处理工具
2. 理解"输入数据→处理逻辑→输出结果"的完整链路
3. 根据实际需求调整处理规则

### 任务说明

#### 核心任务：考勤数据处理与工资计算

给定一份考勤记录CSV和工资参数，写一个工具自动计算每人的应发工资，生成工资单。

#### 分层要求

| 等级 | 完成标准 |
|:---:|:---|
| L1 能用 | 用Claude Code跑通工具，输入考勤CSV，输出每人的工资结果 |
| L2 能调 | L1 + 修改工资计算规则（如调整加班费倍率、增加全勤奖）；能解释数据处理的每个步骤 |
| L3 能交付 | L2 + 输出格式化的工资单（CSV或Excel）；添加数据校验（如检测异常考勤记录）；工具带使用说明 |

### 教学流程

| 时间 | 内容 |
|:---:|:---|
| 0-5分钟 | 展示任务：原始考勤CSV → 运行工具 → 输出工资单。讲解数据处理的通用模式 |
| 5-10分钟 | 学员用中文拆解处理步骤：读取考勤→统计出勤→计算工资→输出结果 |
| 10-35分钟 | 学员用Claude Code独立开发，讲师巡回指导 |
| 35-45分钟 | 2-3位学员展示，对比不同人的实现方式 |

### 教学提示

- 本节比第9节复杂，学员可能会卡在"怎么把需求拆成步骤"上。引导他们先列步骤再编码
- 工资计算规则用学员熟悉的场景（基本工资+加班费-迟到扣款），提高代入感
- 如果有学员提前完成L1，引导他们加功能：比如生成图表、异常值检测
- 数据处理是所有开发工作的基础技能，强调"这个模式以后会反复用到"

### 所需材料/设备

- [ ] 每人一台电脑（WSL + Python + Claude Code）
- [ ] 模拟考勤数据CSV（每台电脑一份）
- [ ] 工资参数说明（打印或投影）
- [ ] 投影仪
