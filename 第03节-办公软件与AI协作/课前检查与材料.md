> 本清单供讲师在上课前 20 分钟完成。所有检查项完成后打勾 `[x]`。

## 一、环境检查（课前20分钟）

### 硬件与网络（5分钟）

- [ ] **投影仪/大屏幕**：开机，连接讲师电脑，画面正常显示
- [ ] **讲师电脑**：浏览器可用，Word/WPS可用，Excel可用
- [ ] **学员电脑**：逐台确认以下内容——
  - [ ] 电脑已开机、已联网
  - [ ] Word或WPS已安装并能正常打开（新建空白文档测试）
  - [ ] Excel已安装并能正常打开（新建空白工作簿测试）
  - [ ] 浏览器可用，能访问AI工具网页
  - 如果某台电脑Word/WPS缺失 → 见下方"备用方案A"
- [ ] **网络速度测试**：在讲师电脑上打开AI工具网页，确认能正常对话、响应在5秒内

### 软件环境确认（5分钟）

- [ ] **Word/WPS版本确认**：检查讲师电脑和学员电脑的版本是否一致。如果混用（部分Word、部分WPS），在教案中注意提示两者的菜单差异
- [ ] **PDF导出测试**：在讲师电脑上新建一个Word文档，尝试导出为PDF，确认流程畅通
  - WPS免费版如有PDF导出限制 → 测试"文件→另存为→PDF"或"打印→Microsoft Print to PDF"作为替代方案
- [ ] **AI工具可用**：打开学员使用的AI工具网页，发送一条测试消息，确认能正常回复
- [ ] **Excel公式测试**：在讲师电脑Excel中输入 `=SUM(1,2)` 确认返回3，公式功能正常

### 环境布置（3分钟）

- [ ] **座位安排**：学员之间留有适当间距，讲师和助教能在走道中巡场
- [ ] **投影画面**：打开Word，输入几行文字，确认从最后一排能看清工具栏和文字内容
- [ ] **讲师站位**：站在投影屏幕旁边，不挡住画面，同时能操作电脑

**排版要点提示（投影在样例旁边）：**

```
简历排版四步走：
1. 标题"个人简历" → 加粗 + 居中 + 大号字（小二/18号）
2. 正文字体统一 → 宋体或微软雅黑，小四或12号
3. 行距设为1.5倍 → 段落设置中修改
4. 完成后导出PDF → 文件 → 导出/另存为 → PDF格式
```

### AI提示词参考（练习环节投影展示）

```
练习任务一 —— AI生成简历内容

请在AI对话框中输入类似以下内容（根据自己情况修改）：

"请帮我写一份简历，包括以下内容：
- 姓名：（你的名字或假名）
- 联系电话：138-0000-0000
- 求职意向：仓库管理员
- 教育背景：XX学校 XX专业 20XX年毕业
- 工作经历：做过2年电子厂操作员，1年外卖骑手
- 请写一段80字左右的自我评价，语气真诚，不要夸大

请直接给我文字内容，不要加格式。"

注意：AI生成后，先检查内容是否真实——
AI可能编造你没有的技能或经历，假的必须删掉！
确认无误后再复制到Word排版。
```

```
练习任务二 —— Excel月度支出表

在Excel中录入以下格式的数据（内容自己编）：

| 日期 | 支出项目 | 金额 |
| 2月1日 | 房租 | 800 |
| 2月5日 | 吃饭 | 450 |
| 2月10日 | 交通 | 100 |
| 2月15日 | 话费 | 50 |
| 2月20日 | 买衣服 | 200 |

然后：
- 用 =SUM(C2:C6) 计算总支出
- 用 =AVERAGE(C2:C6) 计算平均每笔支出
```

### AI写公式环节——投影展示的提示词

```
AI写公式示范

在AI对话框输入：

"我在Excel里有一张工资表。B列是月薪，C列是绩效奖金，
D列是合计。数据从第2行开始，到第4行。
请帮我写一个Excel公式，在E列计算每个人需要交的
个人所得税，假设税率统一按3%计算。"

AI给出公式后：
1. 先读懂公式在算什么
2. 粘贴到Excel中
3. 拿一行数据手算验证（比如5800×3%=174）
4. 确认无误后用填充柄拖拽到其他行
```

## 四、备用方案

### 备用方案A：个别学员电脑缺少Word/WPS或Excel

**现象**：1-2台电脑没有安装Word/WPS或Excel。

**处理步骤**：

1. 尝试快速安装WPS（搜索"WPS官网"下载，安装约5分钟）
2. 如果安装时间来不及：
   - 让该学员使用**在线文档工具**替代：腾讯文档（docs.qq.com）或石墨文档（shimo.im），浏览器中直接打开即可
   - 腾讯文档同时支持文档和表格功能，可以替代Word和Excel
3. 如果在线工具也不行，安排该学员与旁边同学**共用一台电脑**
4. 课后在该电脑上安装WPS

### 备用方案B：AI工具无法访问

**现象**：全场或部分学员的AI工具网页打不开或响应极慢。

**处理步骤**：

1. 先检查是否是网络问题（打开其他网页测试）
2. 如果是AI工具本身的问题：
   - 尝试换一个AI工具（比如平时用的打不开，换另一个网页版AI）
   - 如果所有AI工具都不可用：
     - **Word部分不受影响**，排版操作正常进行
     - **AI生成内容环节改为手动输入**：讲师把一段准备好的简历文字投影出来，学员手动输入到Word中再排版（效果一样练到了排版能力）
     - **AI写公式环节改为讲师直接教公式**：讲师直接告诉学员公式 `=D2*0.03`，学员自己输入
     - 练习环节的AI生成部分同样改为手动输入讲师提供的参考文字

**准备的备用简历文字（AI不可用时投影给学员抄写）：**

```

姓名：张三
联系电话：138-0000-0000

教育背景
2020.09 - 2023.06  XX职业技术学院  物流管理专业

工作经历
2023.07 - 2025.06  XX电子厂  流水线操作员
2025.07 - 2026.01  某外卖平台  外卖骑手

自我评价

熟悉库存管理基本流程。能吃苦耐劳，服从安排，
有较强的责任心和团队协作精神。
做事踏实，乐于接受新知识和新挑战。
```

### 备用方案C：投影仪/大屏幕故障

**现象**：投影仪不亮，或连接不上讲师电脑。

**处理步骤**：

1. 检查连接线是否插紧，切换输入源
2. 如果3分钟内无法解决：
   - 5人以内小班：让学员围过来看讲师电脑屏幕
   - 超过5人：讲师走到每个学员旁边，在学员电脑上演示关键步骤
   - 沈林伟（助教）配合，一人讲一人在学员间巡场指导
3. 本节课操作性强，投影故障影响比第1节课小——重点在学员自己动手

### 备用方案D：WPS导出PDF时弹付费提示

**现象**：WPS免费版导出PDF时要求付费或开会员。

**处理步骤**：

1. 尝试"文件→另存为"，在文件类型下拉菜单中选择PDF格式
2. 如果另存为也不行，用"打印"功能：
   - 点击"文件→打印"
   - 打印机选择"Microsoft Print to PDF"（Windows自带）
   - 点打印，选保存位置，即可生成PDF
3. 如果以上都失败，让学员先完成排版，PDF导出课后解决

## 讲师备忘

- 本节课是第一次使用同伴教学的"学员回顾"机制。第一次可能不太顺畅，学员会紧张，这是正常的。多鼓励，少纠正。重点是让他们开口，而不是讲得多完整。
- 本节课信息密度较高（Word + Excel + AI协作），节奏要紧凑但不能赶。如果时间不够，优先保Word排版和AI协作模式，Excel的排序筛选可以简化。
- **三问法轻触回调**：在AI生成简历内容后，自然地提一句"AI为了讨好你会编东西，简历里的经历你自己最清楚是真是假"。不需要重新讲三问法，点到即止。在AI写公式环节，手算验证时可以说"这就是信但验"。
- **场景延续**：练习场景延续了前两节课的"找仓库管理员工作"主线（第1节搜职位→第2节AI写简历→第3节排版简历→导出PDF）。
- 简历排版练习是本节课的核心闭环（AI生成→检查内容→Word排版→PDF导出），一定要让每个学员完成这个流程。
- 沈林伟作为助教重点负责巡场和个别辅导，特别关注Excel部分有困难的学员。
