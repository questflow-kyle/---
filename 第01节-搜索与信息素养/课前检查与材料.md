# 第1节：搜索与信息素养 —— 课前检查与材料

## 课前15分钟检查清单

> 讲师在上课前15分钟到达教室，按以下顺序逐项检查。每完成一项打勾。

### 硬件与网络（5分钟）

- [ ] **投影仪/大屏幕**：开机，连接讲师电脑，画面正常显示
- [ ] **讲师电脑**：打开浏览器，确认能正常上网
- [ ] **学员电脑**：逐台确认已开机、已联网、浏览器可用
  - 逐台打开 baidu.com，确认能正常加载搜索页面
  - 如果有电脑无法联网 → 见下方"备用方案A"
- [ ] **网络速度测试**：在讲师电脑上搜索一个词，确认结果在3秒内加载完成
  - 如果加载缓慢 → 见下方"备用方案B"

### 教学材料确认（5分钟）

- [ ] **AI工具可用**：在讲师电脑上打开AI工具（网页版），确认能正常对话
  - 在AI工具中输入"你好"，确认能收到回复
  - 记下AI工具的网址，准备写在投影上让学员输入
  - 如果AI工具不可用 → 见下方"备用方案C"
- [ ] **Boss直聘截图**：提前搜索"成都 仓库管理员"，截好搜索结果页面的截图（备用，防止现场需要登录）
- [ ] **天眼查截图**：提前搜索一个公司名，截好天眼查页面截图（备用）
- [ ] **投影内容准备**：确认以下内容可以快速切换展示（见本文档"投影素材"部分）
  - 渠道对照表
  - 关键词提取练习题
  - 招聘帖案例
  - 信息判断三问
  - 平台陷阱速览表
  - 综合练习任务

### 环境布置（3分钟）

- [ ] **座位安排**：学员之间留有适当间距，讲师能在走道中巡场
- [ ] **投影画面**：字体大小从最后一排能看清（打开百度首页，问最后排的人能不能看清搜索框里的字）
- [ ] **讲师站位**：站在投影屏幕旁边，不挡住画面，同时能操作电脑

### 最终确认（2分钟）

- [ ] 教案已打开或打印，放在手边可随时查看
- [ ] AI工具网址已写在投影上或白板上
- [ ] 手机设为静音
- [ ] 确认上课时间，准备开始

## 投影素材

> 以下内容在教案对应环节需要投影展示。提前打开或准备好，确保上课时能快速切换。

### 素材1：渠道对照表（环节一使用）

```
你要干什么？                → 去哪里找？
──────────────────────────────────────────
找工作                      → Boss直聘、前程无忧、智联招聘
查一个公司靠不靠谱          → 天眼查、企查查
查政策法规（工资、社保）    → 政府官网（人社局、社保局）
买东西看评价                → 淘宝/京东 + 小红书对比
学一个技能/查一个知识       → 搜索引擎（百度/必应）+ AI
看新闻时事                  → 多个新闻源交叉对比
```

### 素材2：关键词提取练习题（环节二使用）

```
把下面的问题变成搜索关键词，在百度搜一下：

1. 我在成都上班，公司没给我交社保，这合不合法？
   → 参考关键词：成都 公司 不交社保 违法

2. 我想学Python编程，有什么免费的入门教程？
   → 参考关键词：Python 入门 免费教程

3. 我的电脑最近开机特别慢，是怎么回事？
   → 参考关键词：电脑 开机慢 原因
```

```
福利：包吃住，五险一金，月休6天
联系人：李经理 138XXXX0000

```

### 素材4：信息判断三问（环节三使用）

```
信息判断三问：

① 谁在说？
   → 这个人/账号是谁？亮了身份没有？有没有公司或机构背书？
   → 匿名的 < 实名的 < 有组织背景的

② 能不能验证？
   → 它说的数据、事实，你能不能从另一个渠道查到？
   → 至少用两个渠道交叉验证（搜索引擎 + AI + 官方网站）

③ 动机是什么？
   → 这个人为什么要发这个信息？他能从中得到什么？
   → 免费的东西往往最贵——想想他靠什么赚钱
```

### 素材5：平台陷阱速览表（环节三使用）

```
平台              常见陷阱                        怎么应对
────────────────────────────────────────────────────────────
百度搜索          前几条是广告（标了"广告"二字）    看清有没有"广告"标签，往下翻
Boss直聘          虚假高薪/中介冒充直招            查公司信息（天眼查），注意是否中介
小红书            收费推广伪装成真实分享            看评论区真实反馈，多个帖子对比
淘宝              刷好评/虚假销量                  看追评和差评，别只看好评
微信群/朋友圈     无法验证来源的转发信息            追问原始出处，找不到出处就不信
```

### 素材6：综合练习任务（环节四使用）

```
【综合练习】

你在微信群里看到有人转发了一条消息：

"最新消息！2026年社保缴费基数上调，
 打工人到手工资要减少200-300元！赶紧看看你的工资条！"

请你用今天学的方法验证这条信息：

1. 追问三个问题：
   - 谁在说？（消息的原始来源是什么？）
   - 能不能验证？（社保基数是否真的上调了？到手工资是否真的减少？）
   - 动机是什么？（为什么有人要转发这种消息？）

2. 至少用两个渠道查证：
   - 搜索引擎：试试 "成都 2026 社保缴费基数 site:gov.cn"
   - AI工具：问AI "2026年成都社保缴费基数调整了吗？对到手工资有什么影响？"
   - 也可以用其他你觉得合适的渠道

3. 得出你的结论：这条消息是真的、假的、还是部分真实但有夸大？

时间：8分钟。做完的同学帮一下旁边还没做完的。
```

## 备用方案

### 备用方案A：个别学员电脑无法联网

**现象**：1-2台电脑连不上网络，其他电脑正常。

**处理步骤**：

1. 先尝试基本排查：
   - 检查Wi-Fi是否已连接（看任务栏网络图标）
   - 断开Wi-Fi重新连接
   - 重启浏览器
2. 如果3分钟内无法解决：
   - 安排该学员与旁边的同学**共用一台电脑**
   - 两人一起操作，一个搜一个记，效果可能反而更好
3. 课后再排查这台电脑的问题

### 备用方案B：全场网络缓慢或不稳定

**现象**：所有电脑加载搜索结果都很慢（超过5秒）。

**处理步骤**：

1. 尝试减少同时上网的设备数量：
   - 让学员关闭不必要的网页标签
   - 关闭后台可能占网络的程序
2. 如果网络依然很慢但还能用：
   - 减少让学员动手搜索的次数
   - 改为**讲师演示为主**：讲师在自己电脑上搜，投影给大家看
   - 练习环节改为：每次只让1-2个学员搜索并分享屏幕，其他人观察学习
3. 如果网络完全断了：
   - 用讲师手机开热点，只连讲师电脑，讲师演示
   - 教学内容不变，但所有练习改为讲师演示 + 学员口头回答"你会用什么关键词？""你觉得这条信息靠谱吗？为什么？"
   - 下课时给学员布置课后练习："回去用自己的手机把今天的综合练习做一遍"

### 备用方案C：AI工具无法使用

**现象**：AI工具网页打不开或无法对话。

**处理步骤**：

1. 如果只是加载慢：等一下，或者刷新页面
2. 如果完全无法使用：
   - **不影响核心教学**。本节课AI只在两个地方出现：环节三案例分析中验证工资水平、环节四综合练习中作为渠道之一
   - 环节三：讲师口头告诉学员AI给出的参考数据即可（"如果问AI，它会告诉你成都仓库管理员的平均工资大概4000-6000"）
   - 环节四：让学员用搜索引擎+政府网站两个渠道完成练习，跳过AI渠道
   - 告诉学员："AI工具今天打不开，下节课我们会专门用到它，到时候再体验"

### 备用方案D：投影仪/大屏幕故障

**现象**：投影仪不亮，或者连接不上讲师电脑。

**处理步骤**：

1. 检查连接线是否插紧，切换输入源
2. 如果3分钟内无法解决：
   - 让学员围过来看讲师电脑屏幕（适用于5人以内小班）
   - 或者讲师走到每个学员旁边，在学员电脑上演示关键步骤
3. 需要投影的内容（渠道对照表、三问法等），改为口头讲述或写在白板上

### 备用方案E：学员人数超出预期

**现象**：来了超过10个人，电脑不够。

**处理步骤**：

1. 两人一台电脑，**一人操作一人观察，中间换人**
2. 练习环节时间适当延长，确保每人都有动手机会
3. 如果超过15人，考虑分两批上课（与组织者沟通）

## 讲师备忘

- 这是第一节课，学员可能紧张或者不好意思提问。主动走到学员身边看他们的屏幕，轻声指导，降低他们开口的门槛。
- 第一节课的节奏宁可慢一点，让每个人都跟上，不要赶进度。少讲一个知识点没关系，让学员建立"我能学会"的信心最重要。
- 如果有学员明显基础更好（比如已经会用搜索技巧），可以在练习环节请他/她帮助旁边的同学，提前为后续的"同伴教学"机制做铺垫。
- 本节课AI只是"点到为止"的体验，不需要深入讲解AI的用法。如果学员好奇追问，引导到第2节："下节课我们专门学AI，今天先体验一下。"
- 信息判断三问（谁在说、真不真、为什么）是全课程的核心思维框架，后续课程会反复调用。本节课要确保每个学员都理解并记住这三个问题。
