# 第1节：搜索与信息素养 —— 课前检查与材料

## 课前15分钟检查清单

> 讲师在上课前15分钟到达教室，按以下顺序逐项检查。每完成一项打勾。

### 硬件与网络（5分钟）

- [ ] **投影仪/大屏幕**：开机，连接讲师电脑，画面正常显示
- [ ] **讲师电脑**：打开浏览器，确认能正常上网
- [ ] **学员电脑**：逐台确认已开机、已联网、浏览器可用
  - 逐台打开 baidu.com，确认能正常加载搜索页面
  - 如果有电脑无法联网 → 见下方"备用方案A"
- [ ] **网络速度测试**：在讲师电脑上搜索一个词，确认结果在3秒内加载完成
  - 如果加载缓慢 → 见下方"备用方案B"

### 教学材料确认（3分钟）

- [ ] **搜索实战题目**：已准备好5道搜索题，可以投影展示（见本文档"所需材料"部分）
- [ ] **真假信息辨别素材**：已准备好2条信息（一真一假），可以投影展示（见本文档"所需材料"部分）
- [ ] **高级搜索技巧速查表**：是否需要打印分发？（可选，不是必须）
  - 如果决定打印，确认打印份数 = 学员人数

### 环境布置（2分钟）

- [ ] **座位安排**：学员之间留有适当间距，讲师能在走道中巡场
- [ ] **投影画面**：字体大小从最后一排能看清（打开百度首页，问最后排的人能不能看清搜索框里的字）
- [ ] **讲师站位**：站在投影屏幕旁边，不挡住画面，同时能操作电脑

### 最终确认（2分钟）

- [ ] 教案已打开或打印，放在手边可随时查看
- [ ] 手机设为静音
- [ ] 确认上课时间，准备开始

## 教学素材（可投影展示）

### 搜索实战题目（任务一使用）

```
搜索实战练习 —— 用精准关键词找到答案

1. 成都市2026年最低工资标准是多少？
2. 劳动法规定试用期最长多久？
3. 在成都办理居住证需要什么材料？
4. 工伤鉴定去哪个部门申请？
5. 如何查询个人社保缴费记录？

要求：用关键词搜索，不要输入整句话。
目标：在搜索结果第一页找到答案。
```

### 真假信息辨别素材（任务三使用）

```
请判断以下两条信息哪条是真的、哪条是假的，并说出你的判断依据。

信息A：2026年起，全国统一规定每周工作时间不得超过40小时。

信息B：2026年起，所有企业必须给员工缴纳五险二金
      （五险 + 住房公积金 + 企业年金）。

提示：用今天学的搜索技巧去查证。
```

## 备用方案

### 备用方案A：个别学员电脑无法联网

**现象**：1-2台电脑连不上网络，其他电脑正常。

**处理步骤**：

1. 先尝试基本排查：
   - 检查Wi-Fi是否已连接（看任务栏网络图标）
   - 断开Wi-Fi重新连接
   - 重启浏览器
2. 如果3分钟内无法解决：
   - 安排该学员与旁边的同学**共用一台电脑**
   - 两人一起操作，一个搜一个记，效果可能反而更好
3. 课后再排查这台电脑的问题

### 备用方案B：全场网络缓慢或不稳定

**现象**：所有电脑加载搜索结果都很慢（超过5秒）。

**处理步骤**：

1. 尝试减少同时上网的设备数量：
   - 让学员关闭不必要的网页标签
   - 关闭后台可能占网络的程序
2. 如果网络依然很慢但还能用：
   - 减少让学员动手搜索的次数
   - 改为**讲师演示为主**：讲师在自己电脑上搜，投影给大家看
   - 练习环节改为：每次只让1-2个学员搜索并分享屏幕，其他人观察学习
3. 如果网络完全断了：
   - 用讲师手机开热点，只连讲师电脑，讲师演示
   - 教学内容不变，但所有练习改为讲师演示 + 学员口头回答"你会用什么关键词？"
   - 下课时给学员布置课后练习："回去用自己的手机把今天的5道搜索题做一遍"

### 备用方案C：投影仪/大屏幕故障

**现象**：投影仪不亮，或者连接不上讲师电脑。

**处理步骤**：

1. 检查连接线是否插紧，切换输入源
2. 如果3分钟内无法解决：
   - 让学员围过来看讲师电脑屏幕（适用于5人以内小班）
   - 或者讲师走到每个学员旁边，在学员电脑上演示关键步骤
3. 演示效果会打折扣，但不影响教学进行——重点在于学员自己动手练

### 备用方案D：学员人数超出预期

**现象**：来了超过10个人，电脑不够。

**处理步骤**：

1. 两人一台电脑，**一人操作一人观察，中间换人**
2. 练习环节时间适当延长，确保每人都有动手机会
3. 如果超过15人，考虑分两批上课（与组织者沟通）

## 讲师备忘

- 这是第一节课，学员可能紧张或者不好意思提问。主动走到学员身边看他们的屏幕，轻声指导，降低他们开口的门槛。
- 第一节课的节奏宁可慢一点，让每个人都跟上，不要赶进度。少讲一个知识点没关系，让学员建立"我能学会"的信心最重要。
- 如果有学员明显基础更好（比如已经会用高级搜索），可以在练习环节请他/她帮助旁边的同学，提前为后续的"同伴教学"机制做铺垫。
